# Aparts — каталог недвижимости

Личный сайт на **HTML + CSS + JavaScript** (как [gran-pc](https://github.com/absolutepc/gran-pc)).

## Возможности

- Раздел **ЖК и МФК** — жилые и многофункциональные комплексы с общим количеством квартир и разбивкой по типам
- Раздел **Коммерческая недвижимость** — площадь (м²)
- **Фильтры** — по площади (м²), типу планировки и району (для ЖК/МФК)
- **Админ-панель** — добавление, редактирование и удаление объектов, загрузка фото
- Данные хранятся в `localStorage` браузера

## Поля для ЖК и МФК

| Поле | Описание |
|------|----------|
| Всего квартир | Общее количество квартир в комплексе |
| Однокомнатные | Количество однокомнатных |
| Двухкомнатные | Количество двухкомнатных |
| Трёхкомнатные | Количество трёхкомнатных |
| Евродвушки | Количество евродвушек |
| Площадь от / до | Диапазон площади квартир в комплексе, м² |

## Запуск

Сайт статический, **Node.js не нужен**.

### Вариант 1 — просто открыть файл

Откройте `index.html` в браузере двойным кликом.

### Вариант 2 — через VS Code (рекомендуется)

1. Откройте папку проекта в VS Code
2. Установите расширение **Live Server**
3. Правый клик по `index.html` → **Open with Live Server**

### Вариант 3 — через Python

```bash
python -m http.server 8000
```

Откройте http://localhost:8000
